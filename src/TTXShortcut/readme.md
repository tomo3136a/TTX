# Tera Term ショートカット作成プラグイン

## 概要

メニューから Tera Term を起動するショートカットを作成します。

## 対応バージョン

以下のバージョンで動作を確認

  Tera Term v4.96, v4.98, v4.104, v4.105

## 使い方

1. プラグインをビルドします。
2. Tera Term 実行環境 (ttermpro.exeがあるフォルダ) にビルドした TTXShortcut.dll をコピーします。
3. ショートカット作成を行うときは、メニュー[ファイル-ショートカット作成] を選択します。
4. 「名前」にショートカット名を指定します。
5. 必要に応じてオプションを指定します。\
   (ショートカット起動時に現在の設定ファイルを読み込みます。設定ファイル欄のチェックを外せは起動時読み込みはしません。)
6. 出力先を指定します。(初期値の出力先はデスクトップ)

## ビルド

cmake でビルドする場合

```bat
mkdir build
cd build
cmake -A Win32 ..
cmake --build . --config Release
```
