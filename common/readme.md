# TTX 共通化ライブラリ

TTX 作成を簡単にするために共通機能を纏めました。

次の機能を有しています。

* TTXサポート
  * TTX が無視する対象か確認(TTXIgnore)
  * コマンドラインを解析(TTXGetParam)
  * UI の言語番号(1=English, 2=Japanese, ...)を取得(UILang)
  
* 文字列処理
  * 連結文字列処理関数(StrSet*)
  * 文字列リスト処理関数(*StringList)
  * パス文字列処理(\*Path*)

* 設定ファイル読み書き

* ウインドウ制御
  * コンポーネントのサイズ調整
  * ウインドウの配置
  * ファイル選択・フォルダ選択
